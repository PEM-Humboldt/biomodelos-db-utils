Stats

Comando para realizar consultas SQL y NoSQL relacionadas con estadísticas de uso de BioModelos.

Examples

Listar descargas

bmdbutils stats downloads

Listar grupos

bmdbutils stats groups

Listar modelos

bmdbutils stats models

Listar usuarios

bmdbutils stats users

Parámetros

Usage: bmdbutils stats [OPTIONS] COMMAND [ARGS]...

  Comando para realizar consultas SQL y NoSQL relacionadas con estadísticas de
  uso de BioModelos.

Options:
  --help  Show this message and exit.

Commands:
  downloads  Obtener estadísticas de descarga de modelos de la plataforma
  groups     Obtener estadísticas de los grupos de usuarios en la plataforma.
  models     Obtener estadísticas de los modelos en la plataforma.
  users      Obtener estadísticas de usuarios registrados en la plataforma.

Consideraciones

  • El comando retorna resultados históricos.

Comportamiento del comando

  • El comando consulta estadísticas de uso de la plataforma BioModelos.
  • No modifica ni elimina información.
  • Los resultados se muestran directamente en consola.
